# Homebridge Valetudo

This is a Homebridge plugin that can be used to control your [Valetudo](https://valetudo.cloud) with zero config.

Only the basic feature is supported:

- Start and pause
- Fan speed
- Battery level
- Speaker volume
- Filter indicator
